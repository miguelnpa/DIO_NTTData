Estrutura básica do HTML

voltar

O HTML tem uma estrutura pré-definida composta pelas tags "html", "head" e "body" 

        
            <html>
                <head>
                        // O Head são coisas que vai carregar antes de exibir para o usuário
                    <title> Meu primeiro Html </title>
                </head>
                <body>
                        // Tudo que estiver no corpo é o que de fato vai aparecer para o usuário
                </body>
            </html>

A Tag "HTML" engloba toda a estrutura, enquanto a tag "head" é posicionada antes da tag "body" e é responsável por guardar informações sobre dados da páginas HTML. Por fim a tag body é onde irão as marcações de texto, títulos, parágrafos, etc do HTML.

Falando sobre Tags

voltar

As tags são nomes específicos dentro do HTML que desempenham ações dentro do HTML. A maioria das tags tem abertura e fechamento, exemplo <html></html> porém existem tags que não necessitam desse fechamento a exemplo da tag <img>. As tags podem executar os mais variádos tipos de tarefas como colocar determinado objeto em negrito, colocar uma imagem no HTML, texto, etc.

Atributos das Tags

voltar

Atributos são comandos html reservados para alterar o comportamento de determinada Tag ou colocar uma classe ou identificador nele, por ex. IDs e Classes são facilitadores quando combinados com CSS para modificar visualmente elementos definidos no HTML. Outro atributo importane é o HREF que usado em conjunto com a tag a , por exemplo, permite colocar um endereço de link de uma outra página para o html ou para um elemento na mesma página HTML.

Textos

voltar

Em HTML as tags para modificar textos são de suma importância. Existem tags para marcar título principal da página, demarcar parágrafo, negrito, dentre outras. Seguem exemplos de tipos de Tags voltadas para modificar textos:

  • TAG: h1 - Titulo de maior dentro de uma página
  • TAG: h2 - Subtítulo menor que o h1
  • TAG: p - Parágrafo. Usado toda vez que tiver ponto final.

  • TAG: blockquote - Usada para fazer citação
  • TAG: u - Coloca uma linha embaixo do elemento selecionado
  • TAG: mark - Faz uma marcação no texto
  • TAG: sup - Coloca um símbolo num nível acima da palavra, tipo uma marca registrada

Links

voltar

Links são formas de Navegar dentro da propria página ou ir para outra página na internet. Usa-se a tag "a" (âncora). Eu uso o atributo href para colocar o endereço que quero acessar e o atributo target com valor "blank" ou "self" se quero que abra o link em outra aba ou na mesma.
